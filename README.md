# AquaTech

A mobile application developed for water service technicians 
to manage service requests, digital reports, and customer information.

## Features

- User Authentication
- Service Request Management
- Technician Task Monitoring
- Digital Service Reporting
- Firebase Integration
- Customer ID Upload

## Technologies

- Java
- Android Studio
- Firebase Authentication
- Firebase Realtime Database
- Firebase Storage

## Requirements

- Android 8.0+
- Internet Connection
